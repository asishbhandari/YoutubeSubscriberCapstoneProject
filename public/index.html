Youtube Subscribers Api Endpoints

This is a simple api application offering various endpoints to handle subscriber-related tasks efficiently.Leveraging Node.js with Express.js, it provides a scalable and easy-to-use framework for building RESTful APIs.The application's core functionalities include retrieving subscriber information, fetching names and subscribed channels, and handling requests for specific users by their IDs. In addition, it gracefully handles undefined routes by responding with a 404 status code and an appropriate error message.

  • /Subscribers
  • /Subscribers/names
  • /Subscribers/:id
